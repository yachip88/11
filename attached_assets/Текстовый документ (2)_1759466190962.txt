Алгоритм работы насосов:
Три точки выполнения
положение 0 - ремонт при переводе активного насоса в положение 0 включается остановленый насос
при включении ) на неактивном насосе, активный насос работает по своему стандартному алгоритму

Положение 1 - панель при переводе в это положения активного насоса насос останавливается, пид регулятор обнуляется
при переводе не активного насоса тоже что при 0

Положение 2 пид регулирование


по умолчанию работаем по температуре держим уставку в 50 грС с отсечкой по минимальной частоте 35 гц
Мониторинг высокого давления осуществляется по термометрам на выводе ГВС(если на любом из датчиков давление привышено, насосная группа отключается, запуск возможен только оператором)

алгоритм защиты сухого хода, при наличии сухого хода запускается таймер по умолчанию на 1 минуту насосная группа останавливается, при пропадании сигнала насосная группа запускается, таймер идет, если за время работы таймера получаем еще сработки сухого хода по умолчанию 3 раза, насосная группа выключается, запуск возможен только оператором

по умолчанию насосы работают по наработке с переключением раз в 12 часов 
так же переключение осуществляется в случае наличия ошибок на ПЧ
при выходе из строя датчика температуры так же насосная группа останавливается до момента получения валидных значений
